\# 投資アルゴリズム概要



\## 目的

中長期の株式投資において、

ROEの水準と成長性、利益成長の一貫性を重視した

ファンダメンタル主導の戦略を構築する。



\## 方針

\- データ：J-Quants API

\- DB：SQLite（将来PostgreSQL移行を想定）

\- テクニカル指標は補助

\- 過学習を避け、解釈可能性を重視



\## 注意点

\- 将来予測値は参考情報として扱う

\- 財務指標の意味を重視し、機械的最適化はしない

\## パフォーマンス評価

バックテストによるパフォーマンス評価は、TOPIX指数と比較することにより行います。

TOPIX指数データは J-Quants API を通じて取得し、`index_daily` テーブルに保存されます。
（注: J-Quants APIでは日経平均指数は提供されていないため、TOPIX指数を使用します）

データ更新方法:
\`\`\`
python update_all_data.py --target indices
\`\`\`
